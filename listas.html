Como Criar um Cronômetro em JavaScript

  1. Crie um arquivo HTML com um botão de iniciar e um elemento para exibir o tempo.
  2. No arquivo JavaScript, defina variáveis para minutos, segundos e milissegundos.
  3. Use a função setInterval() para atualizar o tempo a cada 10 milissegundos.
  4. Atualize o conteúdo do elemento HTML com os valores formatados.
  5. Adicione botões para pausar e resetar o cronômetro, controlando o intervalo com clearInterval().
